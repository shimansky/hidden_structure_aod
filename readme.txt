      						Корисна інформація

----------------------------------------------------------------------------------------------------------------------------
    СSS    - тут зберігаються усі стилі оформлення меню, гри, та анімаційні ефекти

    СURSOR - основні типи курсорів

    IMG    - тайлові текстури, зображення анімації персонажів, зброї та предметів

    JS     - увесь код написаний на нативному JavaScript

    MAPS   - файли із двомірними масивами, у вигляді яких можна зберігати мапи ігрових рівнів

    SOUND  - усі звуки

----------------------------------------------------------------------------------------------------------------------------

    index.html - файл для запуску гри

    redactor.html - файл для запуску редактору мап

    room0.html -  файл ігрового рівня

    dev.html - службовий файл із налаштуваннями для розробників
